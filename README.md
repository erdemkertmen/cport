## Installation

```text
git clone https://github.com/haddocking/cport.git
cd cport
pip install -r requirements.txt
python cport.py -h
```

## Example

```text
python cport.py -pdb_file example/1PPE_E.pdb -alignment_format FASTA -sequence_file example/1PPE_msa.fasta
```
